## Packages

This repo uses
[`yarn workspaces`](https://classic.yarnpkg.com/blog/2017/08/02/introducing-workspaces/), which TLDR
means if you want to run a commands in each project then you can either `cd` to that directory and
run the command, or use `yarn workspace [package_name] [command]`.
