# browserslist-config-mamba

The default [Browserslist](https://github.com/browserslist/browserslist) config that targets WebKit
534.48.3 - Safari 5.1 equivalent, for
[Stone Payments - Mamba SDK](https://mambasdk-docs.stone.com.br).

> Browserslist integration is used by
> [@babel/preset-env](https://babeljs.io/docs/en/babel-preset-env#browserslist-integration) in our
> ecosystem.

```bash
yarn add -D browserslist-config-mamba
```
